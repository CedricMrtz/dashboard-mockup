# Equipment Dashboard
This project is a simple dashboard mockup built with Next.js and TailwindCSS.
It demonstrates how data can be displayed and interacted with in a dashboard-style interface.

The application renders product information from a dataset and allows users to filter, search, and add new records dynamically.
## Features
- Dashboard cards showing product statistics
- Data visualization using Recharts
- Dynamic search to quickly find products
- Filtering by category and status
- Add new products directly from the table interface
- Responsive UI built with TailwindCSS
## Tech Stack
- Next.js
- React
- TypeScript
- TailwindCSS
- Recharts
